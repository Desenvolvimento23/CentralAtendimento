# Central de Atendimento Campagnaro

Página institucional para atendimento ao cliente, vendas e pós-venda.

## Estrutura

- `index.html`: página principal.
- `css/`: estilos do site.
- `js/`: interações, busca e perguntas frequentes.
- `assets/images/`: logos e imagens gerais.
- `assets/vendedores/`: fotos dos vendedores.

Para visualizar, abra o arquivo `index.html` no navegador.
